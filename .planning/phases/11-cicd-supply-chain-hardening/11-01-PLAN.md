```yaml
    schedule:

    groups:
      npm-all:
        patterns: ["*"]

  - package-ecosystem: "github-actions"
    directory: "/"
    schedule:
      interval: "weekly"
    open-pull-requests-limit: 5
    labels: ["dependencies", "github-actions"]
    groups:
      actions-all:
        patterns: ["*"]
```

Note: The labels `npm` and `github-actions` may not yet exist in the GitHub repo. GitHub creates `dependencies` by default, but `npm` and `github-actions` are not auto-created. If these labels are absent when the next Dependabot run fires, Dependabot will silently use its default label. This is acceptable — the label config is forward-looking and the labels can be created in the GitHub UI at any time. No checkpoint needed; this is informational only.
